## 申请友链

欢迎任何人申请友情链接。

请相信：尽管下面的要求看起来很苛刻，但实际上是把粗略的内容扩充细化，以保证友链质量。

1. **拒绝以下网站**
   1. 违反人伦、法律、国际公约的；
   2. 检测到存在恶意行为的，或 SSL 证书过期、无效的；
   3. 开盒、挂人等互联网恶俗的，电影、破解软件等版权相关的；
   4. 广告、推销、购物网站、新闻平台（内容的不可审核性）；
   5. 任何人都能发布内容的网站，如论坛、社区、社交媒体平台等（内容的不可审核性）；
   6. 纯转载别人的文章、没有自己原创内容的网站（整合站）；
   7. 协商后仍拒绝添加本站的网站（友链友链，先友后链）；
   8. 出现资源大量 404、页面严重错位、广告横飞等严重影响体验的现象的
2. **有以下情况的，审核可能较为严格**
   <!-- - `.tk`、`.ml`、`.ga`、`.cf`、`.gq`、`.eu.org` 等信誉一般的免费域名； -->
   <!-- - `github.io`、`gitlab.io`、`.vercel.app`、`.pages.dev` 等静态网站托管平台的默认域名； -->
   1. 无 HTTPS 的网站，或使用 IP 访问的；
   2. 24 个月及以上没有新文章或对文章做过修改的；
   3. 12 个月及以上没有任何更新的；
   4. 在无缓存情况下不能在 10s 内加载完成的，或者使用一般设备访问有明显卡顿的；
3. **以下所列不作为审核项目**
   1. 某些国家或地区不能访问（不论是被封禁还是主动拒绝）；
   2. 使用除了中文、英文之外的语言；
   3. 使用免费域名、默认域名；
   4. 适度收费或适度添加非侵入性广告；
   5. 没有 HSTS 或 / 和 ECH；
   6. 非技术类文章 / 负能量文章；

在您添加本站作为友链之后，请按照（不按照也可以）以下格式，在下方的评论区留言名称、描述、链接、头像四条信息：
